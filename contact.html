자주 묻는 질문

문의하기 전에 자주 묻는 질문을 확인해보세요.

서비스 가입은 어떻게 하나요?

홈페이지 상단의 '무료로 시작하기' 버튼을 클릭하여 회원가입을 진행하실 수 있습니다. 가입 즉시 무료 체험 크레딧이 제공되어 서비스를 테스트해보실 수 있습니다.

무료 체험 기간은 얼마나 되나요?

회원가입 시 제공되는 50개의 크레딧으로 무료 체험이 가능합니다. 이는 약 3-5개의 분석을 수행할 수 있는 양입니다. 시간 제한은 없으며 크레딧을 모두 소진할 때까지 사용 가능합니다.

데모는 어떻게 신청하나요?

홈페이지의 '데모 요청' 페이지에서 양식을 작성하여 신청하시거나, info@intrix.ai로 이메일을 보내실 수 있습니다. 영업팀에서 24시간 이내에 연락드려 일정을 조율해 드립니다.

기술 지원은 어떻게 받을 수 있나요?

로그인 후 대시보드의 '지원' 메뉴에서 문의하시거나, support@intrix.ai로 이메일을 보내실 수 있습니다. 또한 웹사이트 우측 하단의 채팅 기능을 통해 실시간 상담도 가능합니다.

API 연동은 지원하나요?

네, 모든 요금제에서 API 연동을 지원합니다. API 문서는 로그인 후 개발자 메뉴에서 확인하실 수 있으며, 연동 관련 문의는 api@intrix.ai로 문의해주세요.
