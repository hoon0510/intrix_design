개인정보 처리방침

Intrix AI가 어떻게 사용자 정보를 처리하는지 알려드립니다.

Intrix AI는 사용자의 개인정보 보호를 중요하게 생각합니다. 본 개인정보 처리방침은 Intrix AI 서비스를 이용하는 과정에서 수집되는 정보와 그 처리 방식에 대해 안내합니다. 서비스를 이용하기 전에 본 방침을 주의 깊게 읽어보시기 바랍니다.

수집 항목

Intrix AI는 서비스 제공을 위해 다음과 같은 정보를 수집합니다:

  • 사용자가 입력한 리뷰 텍스트
  • 제품 또는 서비스명
  • 타겟 고객 정보
  • 목표 포지셔닝
  • 사용 시나리오
  • 시장 유형 선택 정보

저장 정책

Intrix AI는 사용자가 입력한 정보를 서버에 영구적으로 저장하지 않습니다. 입력된 정보는 분석 처리를 위해 일시적으로만 메모리에 보관되며, 분석 결과가 사용자에게 제공된 후 자동으로 삭제됩니다. 이는 사용자의 데이터 프라이버시를 최대한 보장하기 위한 조치입니다.

API 전송

Intrix AI 서비스는 분석을 위해 사용자가 입력한 정보를 Claude 및 GPT API로 전송합니다. 이 과정은 서비스의 핵심 기능을 제공하기 위해 필수적입니다. API 제공업체(Anthropic, OpenAI)는 자체 개인정보 처리방침에 따라 전송된 데이터를 처리할 수 있으며, 이는 Intrix AI의 통제 범위를 벗어날 수 있습니다.

참고: 민감한 개인정보나 기밀 정보는 입력하지 않을 것을 권장합니다. 입력된 모든 정보는 분석 목적으로 외부 API에 전송됩니다.

사용자 책임

Intrix AI가 제공하는 모든 분석 결과와 전략 제안은 참고용으로만 제공됩니다. 이러한 결과를 바탕으로 한 모든 의사결정과 전략 실행은 전적으로 사용자의 책임 하에 이루어져야 합니다. 당사는 제공된 분석 결과의 정확성이나 효과에 대해 어떠한 보증도 제공하지 않습니다.
