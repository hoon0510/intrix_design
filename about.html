Intrix는 무엇을 하나요?

감정 기반 욕구 분석을 통해 고객 반응을 전략으로 자동 변환하는 AI SaaS

사용자 입력 데이터

  • 시장 유형 - 기존 시장인지 신규 시장인지 선택
  • 아이템명 - 제품 또는 서비스명
  • 타겟 고객 - 주요 고객층 정보
  • 사용 시나리오 - 제품/서비스의 사용 상황 설명
  • 목표 포지셔닝 - 원하는 시장 내 위치
  • 고객 리뷰 - 기존 시장의 경우 실제 고객 리뷰나 피드백 (최대 200개)

최소한의 정보만으로 시작할 수 있으며, 리뷰가 많을수록 더 정확한 분석 결과를 얻을 수 있습니다.

AI 분석 항목

감정 흐름 분석

고객 리뷰에서 표현된 감정의 변화와 흐름을 파악하여 제품/서비스 경험의 감정적 여정을 매핑합니다.

욕구 수치화

고객의 기본 욕구(안전, 소속감, 인정, 자아실현 등)를 0-5점 척도로 수치화하여 우선순위를 파악합니다.

커뮤니케이션 스타일

고객이 선호하는 소통 방식과 어조를 분석하여 효과적인 메시지 전달 방식을 제안합니다.

경쟁사 포지셔닝

리뷰에 언급된 경쟁사 정보를 바탕으로 시장 내 차별화 포인트를 식별합니다.

전략 결과 항목

1

전략 요약

핵심 전략 방향과 주요 마케팅 메시지를 간결하게 요약한 전략 개요

2

감정 분석 요약

상위 및 하위 욕구 순위와 각 욕구별 감도를 시각적으로 표현한 분석 결과

3

추천 카피라이팅

분석된 감정과 욕구에 맞춘 3가지 유형의 마케팅 카피 제안

4

브랜딩 전략

Reference Point, Frame Shift, Positioning 문장을 포함한 브랜드 전략 제안

모든 결과는 복사 가능한 형태로 제공되며, 실제 마케팅 활동에 즉시 적용할 수 있습니다.
